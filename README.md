## About

Smart Budget is a simple budget app for zimaos and casaos running on docker

It tracks your spending and will sync your budget to all of your devices automatically 

## How to install

1. make folders and move the files there (u can change the app name if u want)

   /DATA/AppData/smart-budget/app/    ← server/, public/, package.json, package-lock.json go here

   /DATA/AppData/smart-budget/data/   ← leave empty

3. use the [My Budget compose.yaml] on App Store → "Install a Customized App" → Docker Compose tab
4. w8
5. Open http://[your-zimaos-ip]:8090
6. enjoy
